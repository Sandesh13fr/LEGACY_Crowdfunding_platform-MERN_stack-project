# LEGACY Crowdfunding Platform (MERN Stack)

## Description
This repository contains the frontend of a crowdfunding platform built using the MERN stack (MongoDB, Express.js, React, Node.js). The frontend is complete and provides users with an intuitive interface to navigate and manage crowdfunding campaigns.

## Installation
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sandesh13fr/LEGACY_Crowdfunding_platform-MERN_stack-project.git
   cd LEGACY_Crowdfunding_platform-MERN_stack-project
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

## Usage
Once the development server is running, you can access the application at `http://localhost:3000`. The application provides a user-friendly interface to create, manage, and support crowdfunding campaigns.
